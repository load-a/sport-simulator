Data
01 program-mode
	88 default-mode
	88 flag-1
	88 flag-2
	etc.

Procedure
Main.
	perform until quitted
		perform menu-phase
		perform execute-phase
	end

Menu-Phase
	prompt user

	evaluate answer
		when A
			set flag-1
		when B
			set flag-2
		when Q
			set quitted to true
		else
			invalid (warn and continue)
	end

Execute-Phase
	exit if default-mode

	evaluate program-mode
		when flag-1
			perform X
		when flag-2
			perform Y until quitted
		else
			impossible error
	end

	set default mode
